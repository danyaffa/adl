## 🚀 Quick Start

### Prerequisites
- Node.js 16.x or higher
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/adl.git
cd adl

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Start development server
npm start
```
